# E-Commerce Product Catalog

## Overview

E-Commerce Product Catalog is a responsive web application developed using HTML, CSS, and JavaScript. The project allows users to browse products, search items, and add products to a shopping cart with data persistence using LocalStorage.

## Features

* Responsive User Interface
* Product Catalog Display
* Product Search Functionality
* Add to Cart Feature
* LocalStorage Data Persistence
* Mobile-Friendly Design
* Dynamic Product Rendering from JSON Data

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* LocalStorage

## Installation

1. Clone the repository:
   git clone https://github.com/your-username/E-Commerce-Product-Catalog.git

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click on index.html and select "Open with Live Server".

## Usage

* Browse available products.
* Search products using the search bar.
* Add products to the shopping cart.
* Cart information is stored in LocalStorage.
